Chapter 1 · Theory

Backpropagation & Computational Graphs

Understand the exact gradient flow for functions like $f(x,y,z) = (x+y)z$. Master this once and every neural network becomes just a big version of the same idea.

Idea

Computational graph = function broken into tiny ops

A computational graph rewrites a function as a sequence of simple nodes (add, multiply, activation, etc.). Each node has a local derivative. Backpropagation sends gradients from right → left using the chain rule.

Two golden local rules

>Add node $q=x+y$ distributes gradients:
$$\frac{\partial f}{\partial x} = \frac{\partial f}{\partial q},\quad \frac{\partial f}{\partial y} = \frac{\partial f}{\partial q}$$ >Multiply node $f=uv$ swaps inputs as gradients:
$$\frac{\partial f}{\partial u}=v,\quad \frac{\partial f}{\partial v}=u$$
From the lecture

Prof. Godinho de Matos uses exactly this example:
$$q=x+y, \quad f=qz, \quad x=-2,\; y=5,\; z=-4$$

$$q = -2 + 5 = 3, \quad f = 3\cdot(-4) = -12$$
Exam core

Backward pass for $f(x,y,z)=(x+y)z$

Local gradients:

$$\frac{\partial f}{\partial z}=q=3,\quad \frac{\partial f}{\partial q}=z=-4,\quad \frac{\partial q}{\partial x}=1,\quad \frac{\partial q}{\partial y}=1$$

Start with upstream gradient $1$ (at $f$) and move left:

$$\frac{\partial f}{\partial z}=3,\quad \frac{\partial f}{\partial x}=\frac{\partial f}{\partial q}\frac{\partial q}{\partial x}=(-4)\cdot1=-4,\quad \frac{\partial f}{\partial y}=(-4)\cdot1=-4$$

That’s the full backward pass. Chapter 2 drills this pattern until it’s muscle memory.

Chapter 2 · Practice

Gradient Exercises

Start from simple functions and end with a full neuron: affine → sigmoid → squared loss. Try each exercise yourself before opening the answer.

How to attack any exercise >Rewrite the function as small nodes (introduce $q,p,\dots$). >Do a forward pass: compute every intermediate value. >Compute local derivatives at each node. >Run the backward pass using chain rule, right → left.
Exercise 1 · function
Gradients of $f(x,y,z)=xy+z$
x = 3, y = 4, z = 2.  Find: $\partial f/\partial x,\; \partial f/\partial y,\; \partial f/\partial z$.
Solution

Let $p = xy$, $f = p + z$.

$$p = 12,\quad f = 14$$ $$\frac{\partial f}{\partial z}=1,\quad \frac{\partial f}{\partial p}=1,\quad \frac{\partial p}{\partial x}=y=4,\quad \frac{\partial p}{\partial y}=x=3$$ $$\Rightarrow\; \frac{\partial f}{\partial x}=4,\;\; \frac{\partial f}{\partial y}=3,\;\; \frac{\partial f}{\partial z}=1$$
Exercise 2 · same structure as lecture
$f(x,y,z)=(x+y)z$ with new numbers
x = 1, y = 2, z = −3.  Find all: $\partial f/\partial x,\partial f/\partial y,\partial f/\partial z$.
Solution

$q=x+y=3,\; f=qz=-9$.

$$\frac{\partial f}{\partial z}=q=3,\quad \frac{\partial f}{\partial q}=z=-3,\quad \frac{\partial q}{\partial x}=1,\quad \frac{\partial q}{\partial y}=1$$ $$\Rightarrow\; \frac{\partial f}{\partial x}=-3,\; \frac{\partial f}{\partial y}=-3,\; \frac{\partial f}{\partial z}=3$$
Exercise 3 · function
$f(x,y,z)=x(y+z)$
x = −2, y = 3, z = 1.  Find: $\partial f/\partial x,\partial f/\partial y,\partial f/\partial z$.
Solution

Let $s = y+z$, $f=x s$.

$$s=4,\quad f=-8$$ $$\frac{\partial f}{\partial x}=s=4,\quad \frac{\partial f}{\partial s}=x=-2,\quad \frac{\partial s}{\partial y}=1,\quad \frac{\partial s}{\partial z}=1$$ $$\Rightarrow\; \frac{\partial f}{\partial y}=-2,\; \frac{\partial f}{\partial z}=-2$$
Exercise 4 · two adds + multiply
$f(a,b,c,d)=(a+b)(c+d)$
a = 1, b = 2, c = 3, d = 4.  Find: $\partial f/\partial a,\partial f/\partial b,\partial f/\partial c,\partial f/\partial d$.
Solution

Let $p=a+b,\; q=c+d,\; f=pq$.

$$p=3,\; q=7,\; f=21$$ $$\frac{\partial f}{\partial p}=q=7,\quad \frac{\partial f}{\partial q}=p=3$$ $$\Rightarrow\; \frac{\partial f}{\partial a}=\frac{\partial f}{\partial p}\cdot1=7,\; \frac{\partial f}{\partial b}=7$$ $$\frac{\partial f}{\partial c}=\frac{\partial f}{\partial q}\cdot1=3,\; \frac{\partial f}{\partial d}=3$$
Exercise 5 · single neuron
Affine → Sigmoid → MSE loss
$z = w_1x_1 + w_2x_2 + b,\; \hat a = \sigma(z),\; L=(\hat a-y)^2$
$w_1=0.5,\; w_2=-0.3,\; x_1=2,\; x_2=1,\; b=0.1,\; y=1.$
Find $\partial L/\partial w_1,\partial L/\partial w_2,\partial L/\partial b$.
Solution
$$z=0.8,\quad \hat a=\sigma(0.8)\approx0.69,\quad L\approx0.096$$ $$\frac{\partial L}{\partial \hat a}=2(\hat a-y)\approx-0.62$$ $$\frac{\partial \hat a}{\partial z}=\hat a(1-\hat a)\approx0.214$$ $$\frac{\partial L}{\partial z}\approx-0.133$$ $$\frac{\partial L}{\partial w_1}=\frac{\partial L}{\partial z}x_1\approx-0.266$$ $$\frac{\partial L}{\partial w_2}=\frac{\partial L}{\partial z}x_2\approx-0.133$$ $$\frac{\partial L}{\partial b}=\frac{\partial L}{\partial z}\cdot1\approx-0.133$$
Exercise 6 · reverse reasoning
Recover inputs from gradients
$f(x,y,z)=(x+y)z$ with upstream gradient 1 at $f$. You know
$\partial f/\partial x = -5$ and $\partial f/\partial z = 3$.
What can you say about $x,y,z$?
Solution

We know for this graph:

$$\frac{\partial f}{\partial z}=q=x+y,\quad \frac{\partial f}{\partial x}=\frac{\partial f}{\partial q}\frac{\partial q}{\partial x}=z\cdot1=z$$

Given $\partial f/\partial z = 3 \Rightarrow x+y = 3$ and $\partial f/\partial x=-5 \Rightarrow z=-5$.

Any $(x,y)$ with $x+y=3$ is consistent, e.g. $(1,2)$, $(3,0)$, etc.

Chapter 3 · Theory

Epochs, Batch Size & Steps

The professor explicitly said this will be examined: you must be able to compute steps per epoch and total updates, and reason about what happens when you change the batch size.

Definitions >Dataset size $N$: number of training examples. >Batch size $B$: examples per gradient update. >Step / iteration: one weight update (one batch processed). >Epoch: one full pass through all $N$ examples.
$$\text{steps/epoch} = \Big\lceil \frac{N}{B} \Big\rceil, \qquad \text{total updates} = \text{epochs} \times \text{steps/epoch}$$

Ceiling means “round up”: if $N=850$ and $B=100$, then $\lceil 8.5 \rceil = 9$ steps (last batch has only 50 samples).

Chapter 4 · Practice

Batch Size & Epochs – Exam Style

These are exactly the type of numerical questions you can get. Every answer should show the formula, not just the final number.

Exercise B1
Steps and total updates
$N=1000,\; B=50,\; \text{epochs}=20.$
a) Steps/epoch?   b) Total updates?
c) If $B=100$ instead, recompute.
Solution
$$\text{steps/epoch}=\Big\lceil\frac{1000}{50}\Big\rceil=\lceil20\rceil=20$$ $$\text{total updates}=20\cdot20=400$$ For $B=100$: $$\text{steps/epoch}=\Big\lceil\frac{1000}{100}\Big\rceil=10,\quad \text{total}=10\cdot20=200$$
Exercise B2
Non-divisible dataset
$N=850,\; B=64,\; \text{epochs}=15.$
a) Steps/epoch?   b) Total updates?
c) How many samples in the last batch?
Solution
$$\text{steps/epoch}=\Big\lceil\frac{850}{64}\Big\rceil=\lceil13.28\dots\rceil=14$$ $$\text{total updates}=14\cdot15=210$$ Last batch size: $$\text{last} = N - (14-1)\cdot64 = 850 - 13\cdot64 = 850-832 = 18$$
Exercise B3
Compare GD, mini-batch, SGD
Dataset $N=2000$, epochs = 10.
Compare:
(i) $B=N$ (full batch GD)   (ii) $B=32$   (iii) $B=1$ (pure SGD).
For each: steps/epoch and total updates.
Solution
(i) $B=N=2000$:
$$\text{steps/epoch}=\Big\lceil\frac{2000}{2000}\Big\rceil=1,\quad \text{total}=1\cdot10=10$$ (ii) $B=32$:
$$\text{steps/epoch}=\Big\lceil\frac{2000}{32}\Big\rceil=\lceil62.5\rceil=63,\quad \text{total}=63\cdot10=630$$ (iii) $B=1$ (SGD):
$$\text{steps/epoch}=2000,\quad \text{total}=2000\cdot10=20000$$
Exercise B4
Target number of updates
You want exactly 300 gradient updates.
Dataset $N=1200$, batch size $B=40$. How many epochs?
Solution
$$\text{steps/epoch}=\Big\lceil\frac{1200}{40}\Big\rceil=\lceil30\rceil=30$$ $$\text{epochs} = \frac{\text{total updates}}{\text{steps/epoch}} = \frac{300}{30}=10$$
Exercise B5
Effect of halving batch size
$N=5000,\; B=250,\; \text{epochs}=8.$
a) Steps/epoch and total updates?
b) If you halve the batch size to $B=125$ (same epochs), recompute.
c) Qualitatively, what changes besides the counts?
Solution
a) $B=250$:
$$\text{steps/epoch}=\Big\lceil\frac{5000}{250}\Big\rceil=20,\quad \text{total}=20\cdot8=160$$ b) $B=125$:
$$\text{steps/epoch}=\Big\lceil\frac{5000}{125}\Big\rceil=40,\quad \text{total}=40\cdot8=320$$

c) Smaller batches → noisier gradients, more updates, lower memory usage and often slightly better generalisation; but each epoch takes longer.

Exercise B6
Reverse: infer N from logs
Training log: de>epochs = 15, de>total updates = 4800, batch size $B=32$. Assume no partial batches.
a) Steps/epoch?   b) Dataset size $N$?
Solution
$$\text{steps/epoch} = \frac{4800}{15} = 320$$ $$N = \text{steps/epoch} \cdot B = 320 \cdot 32 = 10240$$
